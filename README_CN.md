ArcProof 是一个基于 Arc Testnet 构建的数字服务交付担保平台，面向开发者、设计师、AI 服务提供者、自由职业者及其客户。

在传统数字服务交易中，客户往往担心付款后无法获得承诺的服务，服务提供者也担心完成工作后被拒绝付款。ArcProof 通过智能合约托管 USDC，为双方提供透明、安全、可验证的交易流程。

客户可以创建服务订单，填写服务内容、金额、服务提供者地址和截止日期，并将 USDC 存入智能合约托管。服务提供者完成任务后，提交文件哈希、GitHub Commit Hash、IPFS CID 或其他交付证明。客户确认交付后，智能合约自动将托管资金释放给服务提供者。

如果双方对交付内容存在分歧，客户可以发起争议，资金将继续锁定在合约中，等待后续仲裁处理。仲裁结果可以包括全额退款、全额放款或按比例分账，从而避免任何一方单方面控制资金。

ArcProof 的核心功能包括订单创建、USDC 托管支付、交付证明提交、客户确认放款、争议处理、退款、订单状态追踪和链上交易记录查询。平台适用于 AI 服务、软件开发、网页设计、UI/UX 设计、内容创作、数据分析、咨询服务及其他远程数字服务场景。

项目采用 React、TypeScript、Vite 和 Solidity 构建，支持钱包连接，并部署在 Arc Testnet 上。ArcProof 利用 Arc 的稳定币原生支付能力，实现低成本、透明且可追踪的数字服务结算。

ArcProof 的目标是建立一个面向全球自由职业者、开发者、设计师和 AI Agent 的链上服务担保基础设施，让数字服务交易不再依赖中心化平台，也不再需要在“先付款”与“先交付”之间进行风险选择。
